# Gemini Gems 内训网页课件

这是蓝禾集团内部分享《从会问 AI 到会定义 AI 助手》的静态网页版本。

## 本地打开

直接双击 `index.html`，或在浏览器中打开：

```text
gemini-gems-training/index.html
```

## 操作

- `↑` / `↓` 或 `PageUp` / `PageDown`：切换页面段落
- 右上角“深色”：切换深浅色
- 右上角“打印/PDF”：导出为 PDF

## 发布到 GitHub Pages

把 `gemini-gems-training` 文件夹里的文件放到 GitHub 仓库根目录，或放到仓库的 `docs/` 目录，然后在仓库 Settings -> Pages 里开启 GitHub Pages。

如果放在仓库根目录，发布入口通常是：

```text
https://你的用户名.github.io/仓库名/
```
