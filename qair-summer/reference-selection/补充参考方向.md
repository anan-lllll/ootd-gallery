# Q3 Air Summer 页面参考方向

## 当前精选图的筛选标准

保留方向：
- 城市到轻户外之间的夏日移动生活
- 成年人、有行动感、不过分少女或辣妹
- 可落到官网页面场景：通勤、咖啡/公园、trail、road trip、短途旅行、拍照记录
- 服装偏轻户外、athleisure、休闲机能，而不是硬核登山或纯时装大片

剔除方向：
- 海边辣妹、泳装、过度情绪化摆拍
- 过于校园或低龄的多巴胺画面
- 只表达穿搭好看，但无法承载防护/支架/磁吸/出门移动的画面
- 过于硬核户外，容易让产品承诺变重

## 可继续补充的线上参考

### 1. 城市到户外的轻行动感

参考：lululemon Road To Trail Collection  
链接：https://www.lululemon.co.jp/ja-jp/c/road-to-trail-collection  
可借鉴：从城市跑步/日常运动延伸到 trail 的叙事；适合参考动线、服装和“不是硬核户外”的行动感。

参考：Outdoor Voices Hike Collection  
链接：https://www.fieldmag.com/articles/outdoor-voices-hiker-collection  
可借鉴：轻松、亲近自然、彩色但不幼稚；适合参考人物状态和自然场景里的色彩控制。

### 2. 夏季轻户外装备感

参考：lululemon Hike Collection Lookbook  
链接：https://hypebae.com/2022/7/lululemon-hike-collection-lookbook-where-to-buy  
可借鉴：成年人户外穿搭、岩石/山地/草地场景、机能但不笨重的画面。

参考：Field Mag - lululemon Hike Review  
链接：https://www.fieldmag.com/articles/lululemon-hike-collection-review  
可借鉴：从 studio/城市生活过渡到 trail 的视觉语境，适合我们“城市轻行动人群”的判断。

### 3. 手机壳/随身装备的场景化表达

参考：OtterBox Sole Series  
链接：https://www.otterbox.com/en-us/blog/sole-series.html  
可借鉴：把手机壳讲成 performance + style 的随身装备，尤其是“地铁楼梯到山路”的场景跨度。

参考：OtterBox x Topo Designs  
链接：https://www.otterbox.com/en-us/blog/topo-phone-case.html  
可借鉴：户外复古配色、耐用材料、手机壳和包/配件一起成为日常装备的表达。

参考：Quad Lock limited/cycling/rider scenarios  
链接：https://www.quadlockcase.com/blogs/news/vatternrundan-partners-with-quad-lock  
可借鉴：导航、车载、骑行、移动场景里的手机使用方式；不要照搬硬核运动感，只取“手机是行动工具”的逻辑。

### 4. 更真实的 trail 感，而不是大片式户外

参考：AllTrails Feels Campaign  
链接：https://www.designrush.com/news/alltrails-avocado-coconuts-launch-feels  
可借鉴：真实徒步者、小细节、触感和移动过程，而不是只拍壮阔风景。

## 页面拍摄建议

模特：25-35 岁，干净、健康、有行动感，避免过于网红化或校园化。  
服装：白 T、背心、轻薄衬衫、薄外套、短裤、运动鞋、帆布包、水瓶、墨镜。  
场景：咖啡店外座、城市公园、车内导航、trailhead、湖边/海边但不泳装、短途旅行后备箱。  
手机动作：单手握持、放车载、支架视频/自拍、看地图、拍照、朋友围坐看内容。
